# BasicJira

A production-oriented ASP.NET Core Web API developed during my backend internship at Dias Technology.

## Tech Stack

- ASP.NET Core 9
- Entity Framework Core
- SQL Server
- Clean Architecture
- CQRS & MediatR
- Repository Pattern
- FluentValidation
- JWT Authentication
- RBAC (Role-Based Access Control)
- HashiCorp Vault
- MailKit
- Serilog
- Elasticsearch
- Kibana
